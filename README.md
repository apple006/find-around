## 产品背景

> 在这个快节奏、优体验的社会，我们经常需要接受新的事物和新的环境。那么我们该如何降低熟悉环境的成本，高效快速找到我们的生活和出行所需，这就是Find周边存在的意义。在满足基本功能需求的同时，我们添加了一些便捷操作。专注于周边搜索，针对这种低频但刚需的需求很多人不想专门下个软件。借助小程序用完即走的理念，也许会满足绝大多数人的期望。
